Navigation Tips 'n Tricks

Throughout Windows Master, there are tables such as the one below...

  • Directly beneath the table is a brief description of an action. The context for the description is whichever action is highlited. Since Snap to Bottom Left is highliged, the description is, "Snaps window to the bottom left corner of the monitor."
  • Edit an action by double-clicking on a row; alternativelty, highlight a row and then: press Alt + E, press F2, or click the icon.

In the sequence dialog only, addition, deletion, and editing is allowed...

  • Edit an action or sequence by double-clicking on a row. Alternativelty, highlight a row and then: press Alt + E, press F2, or click the icon.
  • Add an action or sequence by pressing: Alt + A or clicking the icon.
  • Delete an action or sequence by pressing: Alt + D, Delete or clicking the icon.
